=== Apply with Linkedin ===
Contributors: mikejolley
Requires at least: 3.8
Tested up to: 4.0
Stable tag: 2.0.3
License: GNU General Public License v3.0

Add an "Apply with LinkedIn" button to job listings which have an 'email' apply method. Requires API keys from LinkedIn (https://www.linkedin.com/secure/developer).

= Documentation =

Usage instructions for this plugin can be found on the documentation site: [https://wpjobmanager.com/documentation/add-ons/apply-with-linkedin](https://wpjobmanager.com/documentation/add-ons/apply-with-linkedin).

== Installation ==

To install this plugin, please refer to the guide here: [http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation)
